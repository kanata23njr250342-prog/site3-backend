# 3D DESIGNER ROUTE - フロントエンド

Vue 3 + Vite で構築された 3DCG フィードバック機能付きサイト

## セットアップ

```bash
npm install
```

## 開発

```bash
npm run dev
```

## コード品質チェック

### ESLint でコードをチェック
```bash
npm run lint
```

### ESLint で自動修正
```bash
npm run lint:fix
```

## ビルド

```bash
npm run build
```

## 機能

- カテゴリー別の作品表示
- メモ機能（追加、編集、削除）
- メモのドラッグ移動
- メモのリサイズ
- メモの右クリックメニュー
- バックエンド API との連携
